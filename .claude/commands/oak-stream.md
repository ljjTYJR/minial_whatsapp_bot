Stream the OAK-D RGB camera using the oak_view.py script in this project.

Run the following command:

```bash
uv run python oak_view.py
```

The script opens the OAK-D center color camera (CAM_A) at 640x360 and displays the live RGB feed in a window. Press 'q' to quit.

If depthai is not installed, run `uv sync` first.
